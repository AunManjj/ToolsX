How to Customize Colors

To change the color scheme, find the CSS variables section in your HTML file and modify the values:

:root {
  --primary-blue: #0052cc;      /* Change main brand color */
  --primary-dark: #0033a0;      /* Change dark blue */
  --accent-cyan: #00d9ff;       /* Change cyan accent */
  --white: #ffffff;
  --light-gray: #f5f7fb;
  --text-primary: #0a0e27;
  --text-secondary: #5a6b82;
}

✨ 3D Effects & Animations

Key 3D Features

Floating Animation

Subtle up-and-down animation on hero section and logo. Creates a sense of depth and movement.

animation: float 3s ease-in-out infinite

Hover 3D Transform

Cards tilt in 3D space on hover. Uses perspective and rotateY/rotateX for depth effect.

transform: rotateY(8deg)

Glow Pulse

Soft glowing effect that pulses on certain elements. Creates a premium feel.

box-shadow: glow-pulse

Shimmer Effect

Buttons have a shimmer animation on hover. Adds interactivity and polish.

animation: shimmer 2s

Adjusting Animation Speed

To make animations faster or slower, modify the timing values in the CSS:

/* Slow down float animation */
.float-animation {
  animation: float 4s ease-in-out infinite; /* Change 3s to 4s or 5s */
}

/* Speed up hover effects */
.card-3d {
  transition: transform 0.3s ease; /* Change 0.6s to 0.3s */
}

🚀 Deployment & Hosting

Prerequisites

  • Netlify account (free)
  • GitHub/GitLab/Bitbucket repository (recommended)
  • Your domain name (optional)

Step-by-Step Deployment

  1. Prepare Your Files

    Ensure you have: index.html, netlify.toml, build.sh, and any favicon files

  2. Push to Git Repository 
    git add .
git commit -m "Deploy 3D website"
git push origin main
  3. Connect to Netlify

    Go to netlify.com, click "New site from Git", authorize your repository, and connect

  4. Configure Build Settings 
    Build command: bash build.sh
Publish directory: . (current directory)
  5. Set Environment Variables

    In Netlify Site Settings > Build & Deploy > Environment, add:

    ADMIN_PASS = your-secure-password-here
  6. Deploy

    Click "Deploy site" and wait for the build to complete. Your site will be live!

Connect Custom Domain

  1. In Netlify, go to Site Settings > Domain Management
  2. Add your custom domain
  3. Update your domain's DNS records to point to Netlify
  4. Enable automatic HTTPS (Netlify does this by default)

Add Custom Fonts

/* In the head section */
<link href="https://fonts.googleapis.com/css2?family=YOUR-FONT&display=swap" rel="stylesheet"/>

Add Dark Mode

Implement dark mode by adding a media query:

@media (prefers-color-scheme: dark) {
  :root {
    --white: #1a1a1a;
    --light-gray: #2a2a2a;
    --text-primary: #ffffff;
    --text-secondary: #b0b0b0;
  }
}

🔧 Troubleshooting

Favicon Not Appearing

Solution: Clear your browser cache (Ctrl+Shift+Delete), then hard refresh the page. Some browsers cache favicons aggressively.

Animations Stuttering

Solution: Enable GPU acceleration by adding will-change: transform to animated elements. Reduce animation count if on older devices.

Colors Look Different on Mobile

Solution: Some phones have color calibration differences. Test on multiple devices. Ensure sufficient contrast for readability.

Build Fails on Netlify

Solution: Check that build.sh is executable. In your repository: chmod +x build.sh

Modal Not Closing

Solution: Ensure JavaScript is enabled. Check browser console for errors. Verify closeModal() function is not overridden.
